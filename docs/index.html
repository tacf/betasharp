BetaSharp

An enhanced, open-source version of Minecraft Beta 1.7.3, written in C#.

Legacy Beta 1.7.3, Modern Architecture.

Modern C# Codebase

Rewritten from legacy Java to modern C# using the .NET 10 framework for enhanced performance and developer ergonomics.

Performance & Rendering

Features multithreaded chunk meshing, frustum and occlusion culling, and an overhauled OpenGL rendering pipeline.

Microsoft Authentication

BetaSharp comes with a custom launcher that authenticates using your Microsoft account.

Custom Effects

Elevate the visuals without losing the retro charm. Features new graphical additions such as chunk fade and waving leaves.

Quality of Life

Built-in support for high-resolution texture packs, Controllers (Xbox, Playstation, Steam Deck, etc), and more.

Open Source

BetaSharp is completely free and open-source software. The community is welcome to browse the code, report issues, and submit contributions!

BetaSharp requires a legally purchased copy of Minecraft. We do not support or condone piracy. Please purchase Minecraft at minecraft.net.
